Using Bar Charts


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/bar-chart/bar-chart.js"></script>
  </head>

The default name for the bar chart widget is "bar-chart" so the widget manager will look for class names of "bar-chart", "bar-chart-data", and "bar-chart-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
