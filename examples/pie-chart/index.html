Using Pie Charts


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/pie-chart/pie-chart.js"></script>

The default name for the pie chart widget is "pie-chart" so the widget manager will look for class names of "pie-chart", "pie-chart-data", and "pie-chart-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
