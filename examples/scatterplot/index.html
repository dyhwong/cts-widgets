Using Scatterplots


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/scatterplot/scatterplot.js"></script>
  </head>

The default name for the scatterplot widget is "scatterplot" so the widget manager will look for class names of "scatterplot", "scatterplot-data", and "scatterplot-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
