Using Line Graphs


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/line-graph/line-graph.js"></script>
  </head>

The default name for the line graph widget is "line-graph" so the widget manager will look for class names of "line-graph", "line-graph-data", and "line-graph-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
