Using Grouped Bar Charts


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/grouped-bar-chart/grouped-bar-chart.js"></script>
  </head>

The default name for the grouped bar chart widget is "grouped-bar-chart" so the widget manager will look for class names of "grouped-bar-chart", "grouped-bar-chart-data", and "grouped-bar-chart-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
