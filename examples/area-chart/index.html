Using Area Charts


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/area-chart/area-chart.js"></script>
  </head>

The default name for the area chart widget is "area-chart" so the widget manager will look for class names of "area-chart", "area-chart-data", and "area-chart-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
