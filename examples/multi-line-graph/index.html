Using Multi Line Graphs


  <head>
    <!-- load the widget manager -->
    <script src="http://treesheets.org/widgets/widget-manager.js"></script>

    <!-- load the widgets -->
    <script src="http://treesheets.org/widgets/multi-line-graph/multi-line-graph.js"></script>
  </head>

The default name for the multi line graph widget is "multi-line-graph" so the widget manager will look for class names of "multi-line-graph", "multi-line-graph-data", and "multi-line-graph-properties". Properties can be changed by adding an HTML element with the corresponding class name under the properties element.
